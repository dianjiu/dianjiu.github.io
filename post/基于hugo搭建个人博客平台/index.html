基于hugo 搭建的博客系统

安装hugo

win10

1
2

choco install hugo -confirm
choco install hugo-extended -confirm

mac

1

brew install hugo

linux

1

brew install hugo

创建blog

1

hugo new site blog

添加themes

1
2
3
4

cd blog
git init
git submodule add https://github.com/olOwOlo/hugo-theme-even.git themes/even
echo theme = \"even\" >> config.toml
